= AsciiDocumenter.jl

Convert AsciiDoc to Markdown for use with link:https://documenter.juliadocs.org/[Documenter.jl].

== Quick Start

[source,julia]
----
using Pkg
Pkg.add("AsciiDocumenter")
----

[source,julia]
----
using AsciiDocumenter

# Parse AsciiDoc
doc = AsciiDocumenter.parse("""
= My Document

This is *bold* text.

See link:@ref[My Function].
""")

# Convert to Markdown string for Documenter
md = AsciiDocumenter.to_markdown(doc)
----

== Supported Features

[cols="1,1,1"]
|===
| Feature | AsciiDoc Syntax | Documenter Result

| *Cross-Refs*
| `link:@ref[Target]`
| `[Target](@ref)`

| *Math*
| `stem:[x^2]`
| `$x^2$`

| *Docstrings*
| `[source,@docs]`
| ```` ```@docs ````

| *Admonitions*
| `[NOTE]`
| `!!! note`

| *Code*
| `[source,julia]`
| ```` ```julia ````

| *Standard*
| Headers, Lists, Tables
| Standard Markdown
|===
